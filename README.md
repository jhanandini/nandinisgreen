# nandinisgreen - AI Librarian Chatbot

An intelligent, friendly, and witty book recommender system built using the **Gemini 2.5 Flash** model and **Streamlit**. It acts like your supportive literary peer, understanding your mood and serving the perfect book suggestions in Hinglish!

## Features
* **Personalized Recommendations:** Suggests 2-3 amazing books based on your exact vibe or mood.
* **Conversational AI:** No formal or heavy textbook language—talks just like a friend.
* **Secure Setup:** Keeps the API keys completely hidden using environment variables (`.env`).
